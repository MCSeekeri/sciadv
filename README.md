# 介绍
站点使用Hexo搭建,所有内容都针对Hexo及Markdown进行了特别优化。

收录内容包括科学ADV全系列作品(混沌之子,命运石之门,机器人笔记,混沌之脑,超自然9人组和仍在咕咕咕的匿名代码)已被翻译的小说。
同时特别收录"未来同人研究所"撰写的同人小说。
